2. DATA SOURCES
=============================================================================

2.1 House Price Data
    - Source: UK Land Registry Price Paid Data
    - Period: 2022-2024
    - Variables: Transaction ID, Price, Date, Postcode, Town, County

2.2 Broadband Speed Data
    - Source: Ofcom Fixed Broadband Performance Data
    - Variables: Postcode, Average Download Speed, Maximum Download Speed

2.3 Crime Data
    - Source: UK Police Crime Data
    - Period: 2022-2024
    - Variables: Crime Type, Location, LSOA, Month
    - Focus: Drug offences, Vehicle crime, Robbery

2.4 Supporting Data
    - LSOA to Postcode Mapping
    - Population Data (2011 Census)
